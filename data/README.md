## 📊 Dataset

This project uses the Brazilian E-Commerce Public Dataset by Olist, which contains approximately 100,000 real-world orders from 2016–2018.

The dataset includes:

- Customer information  
- Order details and timestamps  
- Payment data  
- Product and seller data  
- Delivery and logistics data  
- Customer reviews  

Source:
https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce
