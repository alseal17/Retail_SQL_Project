# 🛒 Retail E-Commerce SQL Analytics (PostgreSQL)

## ⭐ Project Highlights

- Designed a relational database using PostgreSQL to analyze e-commerce data  
- Performed multi-table joins across customers, orders, products, and payments  
- Analyzed revenue trends, customer behavior, and product performance  
- Evaluated delivery operations and their impact on customer satisfaction  
- Implemented advanced SQL techniques including window functions, ranking, and growth analysis  

---

## 📌 Overview

This project analyzes a real-world Brazilian e-commerce dataset to generate business insights using SQL. The dataset includes customer information, orders, payments, products, and delivery details.

The goal of this project is to simulate real-world data analyst tasks by answering business questions related to revenue, customer behavior, product performance, and operational efficiency.

---

## 🎯 Objectives

- Design and implement a relational database schema  
- Perform data exploration and validation  
- Analyze revenue trends and sales performance  
- Understand customer purchasing behavior  
- Evaluate product performance and pricing  
- Assess delivery performance and customer satisfaction  
- Apply advanced SQL techniques for deeper insights  

---

## 📊 Dataset

This project uses the Brazilian E-Commerce Public Dataset by Olist, which contains approximately 100,000 orders from 2016–2018.

The dataset includes:

- Customer information  
- Order details and timestamps  
- Payment data  
- Product and seller data  
- Delivery and logistics data  
- Customer reviews  

---

## 📁 Project Structure

- schema/ → database table creation scripts  
- data/ → raw dataset (not included)  
- queries/ → SQL analysis scripts  
- reports/ → summary insights and outputs

### 🔥 Advanced SQL Analysis

## 🧠 Key Insights

- Revenue shows consistent growth trends over time, with seasonal fluctuations  
- A small percentage of customers contribute disproportionately to total revenue  
- Certain product categories dominate both sales volume and revenue  
- Delivery delays negatively impact customer review scores  
- Operational performance varies significantly by region
